# Drax Editorial Blog

This is a self-contained Astro editorial blog surface generated by Drax.

It is independent of the customer site's stack. Deploy it locally on the founder VPS as static output mounted under the approved subpath or subdomain.

## Site Identity

- Site name: `__SITE_NAME__`
- Site URL: `__SITE_URL__`
- Description: `__SITE_DESCRIPTION__`
- Mount mode: `__MOUNT_MODE__`
- Base path: `__BASE_PATH__`

Any `NEEDS_DECISION` value must be resolved in the founder docs before production deployment.

## Commands

```bash
npm install
npm run build
npm run preview
```

## Content

Add Markdown or MDX files to `src/content/posts/`.

Required frontmatter:

```yaml
title: "Post title"
description: "Post summary"
publishedAt: 2026-06-06
tags:
  - example
draft: false
```

Optional frontmatter:

```yaml
updatedAt: 2026-06-06
canonicalUrl: "https://example.com/blog/post"
coverImage: "/og/post.png"
```
